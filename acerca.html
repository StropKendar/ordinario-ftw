Acerca de LaptopHub

Sistema de consulta y exploración de laptops desarrollado con tecnologías web.

Descripción del proyecto

LaptopHub es una plataforma web que permite consultar información de más de 100 laptops mediante el uso de archivos XML y JavaScript.

Objetivo

Facilitar la búsqueda, filtrado y consulta de especificaciones de laptops pertenecientes a distintas marcas.

Tecnologías utilizadas

  • HTML5
  • CSS3
  • JavaScript
  • XML
